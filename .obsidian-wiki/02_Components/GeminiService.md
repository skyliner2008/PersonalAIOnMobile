# 🤖 GeminiService

`GeminiService.kt` จัดการการสื่อสารกับ Gemini API ผ่าน REST (Non-Live mode)

## 🎯 หน้าที่หลัก
1. **Chat & Text Generation**: รองรับการแชทแบบปกติและการใช้เครื่องมือ (Function Calling)
2. **Streaming Flow**: ส่งออกคำตอบแบบต่อเนื่อง (Streaming)
3. **Embeddings**: แปลงข้อความเป็นเวกเตอร์เพื่อใช้ใน Semantic Search
4. **File Analysis**: วิเคราะห์ไฟล์แบบ Native (เช่น PDF, Images)
